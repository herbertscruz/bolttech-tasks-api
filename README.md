# Bolttech Tasks API

This is a multi-user task manager API by Bolttech.

## Install

```sh
npm i
npm run db:migration:run
npm run db:seed
npm start
```

## Tools

- [Adminer](http://localhost:8080/?server=db&username=root&db=bolttech_tasks);
